Puntuación de Característica: Tener un valor mínimo en una característica (Fuerza, Destreza, Constitución, Inteligencia, Sabiduría o Carisma).
Ataque Base (AB): Tener un bonificador de Ataque Base mínimo.
Habilidades: Tener un número específico de rangos en una o varias habilidades.
Dotes previas: Poseer una o más dotes específicas, o un número determinado de dotes de un tipo concreto (ej. "2 dotes metamágicas", "cualquier dote de Suerte").
Nivel de Clase: Tener un nivel mínimo en una clase específica (ej. "Nivel 4 de Guerrero", "Nivel 1 de Hechicero").
Nivel de Personaje: Tener un nivel global de personaje mínimo (ej. "Nivel de Personaje 6", "Solo a nivel 1").
Raza o Tipo de Criatura: Pertenecer a una raza concreta (ej. "Elfo", "Gnomo"), a una etnia humana (ej. "Iluskano"), o tener un tipo/subtipo de criatura (ej. "Aberración", "Sangre de Dragón", "Constructo").
Tamaño: Tener una categoría de tamaño específica (ej. "Grande o superior", "Pequeño o menor").
Alineamiento: Tener un alineamiento específico o restringido (ej. "No ser Bueno", "Cualquier alineamiento Caótico").
Magia y Conjuros:
Tener un Nivel de Lanzador (NL) mínimo (Arcano o Divino).
Capacidad para lanzar conjuros de un nivel específico (ej. "Nivel 3").
Capacidad para lanzar conjuros de una escuela o subsusbsistema específico (ej. "Ilusión", "Invocaciones").
Conocer un conjuro específico o tener un descriptor de energía concreto.
Acceso a Dominios de clérigo específicos.
Aptitudes de Clase o Raciales: Poseer una capacidad especial concreta (ej. "Música de Bardo", "Forma Salvaje", "Ataque Furtivo", "Furia", "Expulsar/Reprender muertos vivientes", "Evasión").
Competencias: Tener competencia con tipos de armadura (ligera, media, pesada), escudos o armas específicas/exóticas.
Religión y Organizaciones: Adorar a una deidad tutelar específica, pertenecer a una orden de caballería, logia o culto.
Región: Proceder de una región geográfica específica (común en las dotes regionales).
Rasgos Físicos: Tener características físicas concretas (ej. "Tener alas", "Tener dos o más ojos", "Tener mano de obra").
Condiciones Especiales: Requisitos narrativos o de estado (ej. "Haber realizado un pacto con un diablo", "Haber muerto y sido revivido", "Tener un familiar").
